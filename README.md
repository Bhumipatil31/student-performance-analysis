# Student Performance Analysis

## Objective
To analyze student academic performance and identify key factors affecting outcomes
using exploratory data analysis.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Insights
- Students who completed test preparation courses scored higher
- Parental education level impacts academic performance
- Reading and writing scores are strongly correlated
- Low-performing students can be identified using total score analysis

## Recommendations
- Promote test preparation programs
- Provide academic support to low-performing students
- Introduce mentoring initiatives for academic improvement
